/page

Parameters

/page/:councilperiod/:page

Example

https://dcstat.dccode.gov/page/20/187

Description

Redirects the user to a page in the DCSTAT. [Note: Only Council Period 20 works at the moment.]

/get

Parameters

/get/:measure.pdf

Example

https://dcstat.dccode.gov/get/Law_19-239

Description

Redirects the user to a measure in the DCSTAT, given the measure’s citation. There are four types:

  • Law
  • Act
  • Resolution
  • Proposed_Resolution

/search

Parameters

/search?q=<query>

Example

https://dcstat.dccode.gov/search?q=%22No%20Taxation%20Without%20Representation%22

Description

Returns the search page with the desired search query.

/api/search

Parameters

/api/search?q=<query>

Example

https://dcstat.dccode.gov/api/search/?q=%22Flag%20Amendment%22

Description

Returns a JSON object of search results for a given search query.

/api/measures

Parameters

/api/measures/:measure

Example

https://dcstat.dccode.gov/api/measures/Law_19-239/

Description

Returns a JSON object with a measure’s metadata, including full-text of the measure.
