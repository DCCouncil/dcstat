Internet Archive Accessibility

As part of the Council's commitment to ensure that official electronic legal materials are preserved, the Council has uploaded the Council Period 20 DCSTAT (as well as historical versions from 1975 through 1986) to the Internet Archive. The files can be accessed here.

Authentication

To ensure the authenticity of the documents included on this page, the Council is currently using two methods:

  1. SSL/TLS. A secure method ensuring that transmission of website data requested by a user has not been changed.
  2. SHA Hash Values. Using a cryptographic algorithm (SHA-256), a unique value is calculated based on the contents of a particular file. If the file has been altered in any way, the computed value will be different.

In the coming weeks, the Council will fully implement its new Authentication Service that will allow for Digital Signatures and will allow a user to upload a file to check whether the file is valid.
